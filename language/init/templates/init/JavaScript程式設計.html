本課程培養學生下列知識：
1. 知識：介紹電腦網路及網際網路的發展過程及運作的原理 2. 技能：使學生具備分析網際資訊系統及網路運作架構的能力 3. 態度：培養學生能主動學習及研究網路技術的態度 4. 其他：為後續各種網路應用之資訊系統課程的基礎

1.學生需了解JavaScript的基本語法
2.學生需了解HTML的文件物件模型
3.學生需了解jQuery的基本語法
4.學生需能利用jQuery設計網頁前端的功能

The goal of this course is to provide introductions to JavaScript programming for enhancing the functions of Web pages, including Web page styles, visual effects, interactivity, and animation. At the end of this course, students should be able to: 1.Design and create JavaScript programs that let the user interact with a Web page. 2.Create HTML pages that contain one or more JavaScript functions. 3.Load and display HTML pages that include JavaScript commands. 4.Create JavaScript programs that display input controls on an HTML page and allow the user to submit data to a Web page. 5.Create JavaScript programs that read user input data based on the contents of the data and perform alternate actions.
